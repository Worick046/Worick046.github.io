Terms & Definitions

The following use of 'we', 'our', and 'us' refers to the Rollbot dev team.

The following use of 'the application' refers to the Rollbot.

The following use of 'Roll Names' refers to the user supplied names for custom rolls.

Privacy Policy

Upon explicit consent prompted by the Rollbot, users subject themselves to the following.

Collection and Storing of Data

Rollbot will collect and store the following data.

  • Member id

  • Roll Names

Usage of Data

Data collected is used in the following ways

  • Enabling the use of customized rolls for users

Third Party Data Sharing

Rollbot will not share data to other parties outside of the server in which the Rollbot was used in.

Parties inside the server Rollbot was used in may see the Roll Names you supply to the Rollbot when you invoke the application.
